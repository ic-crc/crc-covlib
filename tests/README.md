# C++ tests

## How to run tests

From a test's directory, you may use one of the following command:
```bash
make [CONFIG={debug|release}] # Build crc-covlib (if required) and test program (default CONFIG is release).

make rebuild [CONFIG={debug|release}] # Rebuild crc-covlib (if required) and test program (default CONFIG is release).

make run # Run the test program once built.

make clean # Delete files generated by the build process or by running the test.
```
